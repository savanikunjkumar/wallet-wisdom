# Wallet Wisdom 💸

**AI-powered expense tracking and forecasting. Take control of your finances with Machine Learning and the Cloud.**

## 📌 About This Project
Hi! I built Wallet Wisdom as part of my Open Source Software Evaluated Course Project. 

Traditional expense trackers only tell you what you've *already* spent. I wanted to build something smarter. By combining a Python web framework with Scikit-Learn's Machine Learning algorithms and a PostgreSQL cloud database, this app actually analyzes your spending habits and predicts your future expenses. 

## ✨ What It Does
* **Cloud Syncing:** Uses Supabase to save and fetch expense data in real-time. No local files to lose.
* **AI Forecasting:** Takes your historical spending data and runs it through a Random Forest Regressor to predict what you'll likely spend next month.
* **Simple Dashboard:** A clean, easy-to-use web interface built completely in Python (no HTML/CSS required).
* **Data Visualization:** Automatically generates charts so you can see your spending trends at a glance.

## 💻 Tech Stack
* **Language:** Python
* **Frontend:** Streamlit
* **Database:** Supabase (Open-source PostgreSQL)
* **Machine Learning:** Scikit-Learn, Pandas, NumPy
